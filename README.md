# Antpocalypse

This is a game for the Fall 2025 Redot Game Jam. In our game you play the role of an ant who's colony just survived an apocalyptic envent, almost destroying the colony. Rebuild your colony from the rubble and bring your fellow ants back from the brink!

## Development Requirements

To work on this game, please use the Redot 4.3.1 game engine.
